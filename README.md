# Cis344-group-project
By Juan Green, Nasir Karim and Aboubacar Tounkara

Art Gallery Management System

This project is a mini web based Art Gallery Management System built using PHP, MySQL, HTML and Css. It allows users to manage artworks, exhibitions, and sales in an organized way.

Features:
- View all artworks with artist details and images
- Search for artworks by title
- Add new artworks
- Delete artworks
- Sell artworks and track sales
- View exhibitions available

Users can add new artworks by entering a title, artist ID, price, and image URL. The artist ID must match an existing artist in the database (1-5). This means you can add any artwork, but it must be associated with one of the artists already stored in the system or if not it will still work but different artist name.

How to Run
1. Start Apache and MySQL
2. Import the artgallerydb.sql file into phpMyAdmin
3. Place the project folder or files inside the htdocs folder in XAMPP


Open your browser and type localhost/cis344project/(or whatever your folder name is)
